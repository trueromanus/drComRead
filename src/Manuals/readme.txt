drComRead
версия 0.14

Программа drComRead для просмотра изображений форматов JPEG,JPEG2000,BMP,GIF,PNG,TIFF,PSD,TGA,PaintBrush PCX,DICOM,DCX,WDP,HDP,WBMP.
Поддерживается масштабирование, выравнивание по ширине и высоте изображения, разделение на альбом и портрет с возможность выставления для каждой своей настройки.
Просмотр изображений в полный экран с возможностью активного перемещения по направляющим окна.
Двухстраничный просмотр изображений (с двухстраничными или нет переходами) в котором доступны все виды масштабирования.
Опции цветокорреции а также визуальный инструмент для редактирования кривых Безьё для текущего изображения с возможностью сохранения в файл.
Чтение изображений из архивов RAR(CBR), ZIP(CBZ), 7ZIP(CB7), TAR(CBT), ARJ(CBA), GZIP.
Возможность загрузки каталогов с индексацией всех архивов в папке (и всех подпапках) и возможностью просмотра их в листинговом порядке.
Возможность поворотов изображений, зеркальное отражение, сохранение в буфер обмена и файл. Сортировка списка изображений и архивов.
Управляющий элемент навигатор для удобных переходов.
"Корешки" в верхнем и нижнем углах экрана для быстрых переходов между страницами.
Всплывающая подсказка при переходах между страницами с помощью колесика мыши.
Сквозные переходы между архивами при навигации по списку изображений.
Диалог истории всех просмотренных архивов и папок с возможностью сортировки и предпросмотра обложек архивов.
Возможность перепаковки архивов а также создания архивов из списка изображений.
Возможность просмотра изображений из интернета или скачивания и запаковки их в архив.
